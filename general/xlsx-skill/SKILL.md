# Excel (XLSX) Output — Zuma Operations

Skill ini digunakan untuk menghasilkan file Excel (.xlsx) yang rapi, berformula, dan siap pakai untuk operasional Zuma.

## Kapan Digunakan

- Output planogram toko ke Excel
- RO Request sheet (Protol + Box + Surplus)
- Laporan sales / inventory untuk management
- Template yang perlu diisi tim (SPG, SPV)
- Konversi data dari DB ke Excel yang readable

---

## Library

| Library | Gunakan untuk |
|---------|--------------|
| `openpyxl` | Formula, formatting, multi-sheet, merge cells |
| `pandas` | Data prep, bulk write, analisis sebelum export |
| `xlsxwriter` | Charts embedded di Excel, advanced formatting |

**Default choice:** `openpyxl` untuk output production. `pandas` untuk data prep.

---

## Prinsip Wajib

### 1. Gunakan Formula Excel, BUKAN hardcode Python result

```python
# ❌ SALAH — hardcode nilai dari Python
total = df['qty'].sum()
sheet['D10'] = total  # hardcoded 500

# ✅ BENAR — biarkan Excel yang hitung
sheet['D10'] = '=SUM(D2:D9)'
```

### 2. Zero Formula Errors

Sebelum deliver, pastikan tidak ada: `#REF!`, `#DIV/0!`, `#VALUE!`, `#N/A`, `#NAME?`

### 3. Zuma Color Convention

| Warna | Hex | Digunakan untuk |
|-------|-----|-----------------|
| Teal | `#002A3A` | Header row, title |
| Green | `#00E273` | Highlight, total row, target achieved |
| White | `#FFFFFF` | Data rows |
| Light gray | `#F5F5F5` | Alternate rows |
| Red | `#FF4444` | Warning, deficit, shortfall |

## Pola Dasar

### Buat File Baru (openpyxl)

```python
from openpyxl import Workbook
from openpyxl.styles import Font, PatternFill, Alignment, Border, Side
from openpyxl.utils import get_column_letter

wb = Workbook()
ws = wb.active
ws.title = "Sheet1"

# Header styling (Zuma teal)
header_fill = PatternFill("solid", fgColor="002A3A")
header_font = Font(bold=True, color="FFFFFF", size=11)

headers = ["Kode Toko", "Nama Toko", "Artikel", "Qty Target", "Qty On Hand", "Selisih"]
for col, h in enumerate(headers, 1):
    cell = ws.cell(row=1, column=col, value=h)
    cell.fill = header_fill
    cell.font = header_font
    cell.alignment = Alignment(horizontal="center", vertical="center")

# Data rows
data = [
    ("JT001", "Toko Surabaya", "ZM-001", 10, 6),
    ("JT002", "Toko Malang", "ZM-001", 8, 8),
]
for row_idx, row_data in enumerate(data, 2):
    for col_idx, val in enumerate(row_data, 1):
        ws.cell(row=row_idx, column=col_idx, value=val)
    # Formula di kolom Selisih (F = D - E)
    ws.cell(row=row_idx, column=6, value=f"=D{row_idx}-E{row_idx}")

# Auto-width kolom
for col in ws.columns:
    max_len = max((len(str(c.value or "")) for c in col), default=10)
    ws.column_dimensions[get_column_letter(col[0].column)].width = min(max_len + 4, 40)

# Freeze header
ws.freeze_panes = "A2"

wb.save("output.xlsx")
```

### Load dan Edit File Existing

```python
from openpyxl import load_workbook

wb = load_workbook("template.xlsx")
ws = wb["RO Request"]

# Edit cell tertentu
ws["B5"] = "Updated value"

# Loop rows dan update
for row in ws.iter_rows(min_row=2, max_row=ws.max_row, values_only=False):
    kode_toko = row[0].value
    if kode_toko == "JT001":
        row[3].value = 15  # Update qty

wb.save("output_updated.xlsx")
```

### Multi-Sheet Workbook (Pola RO Request)

```python
from openpyxl import Workbook

wb = Workbook()

# Sheet 1: Cover
cover = wb.active
cover.title = "Cover"
cover["A1"] = "RO Request — Week 10 2026"
cover["A2"] = "Generated by Iris"

# Sheet 2: RO Protol
protol_ws = wb.create_sheet("RO Protol")
protol_ws.append(["Kode Toko", "Nama Toko", "Artikel", "Ukuran", "Qty RO"])

# Sheet 3: RO Box
box_ws = wb.create_sheet("RO Box")
box_ws.append(["Kode Toko", "Nama Toko", "Artikel", "Ukuran", "Qty RO"])

# Sheet 4: Surplus
surplus_ws = wb.create_sheet("Surplus Pull")
surplus_ws.append(["Kode Toko", "Nama Toko", "Artikel", "Ukuran", "Qty Surplus"])

# Sheet 5: Reference
ref_ws = wb.create_sheet("Reference")
ref_ws["A1"] = "Sumber data: portal.planogram + core.v_stock_on_hand"

wb.save("RO_Request_W10_2026.xlsx")
```

## Formatting Helpers

### Alternate Row Colors (Zebra Stripes)

```python
from openpyxl.styles import PatternFill

fill_light = PatternFill("solid", fgColor="F5F5F5")
fill_white = PatternFill("solid", fgColor="FFFFFF")

for row_idx, row in enumerate(ws.iter_rows(min_row=2), 2):
    fill = fill_light if row_idx % 2 == 0 else fill_white
    for cell in row:
        cell.fill = fill
```

### Conditional Formatting (Selisih Merah/Hijau)

```python
from openpyxl.formatting.rule import ColorScaleRule, CellIsRule
from openpyxl.styles import PatternFill

# Merah kalau negatif (deficit)
red_fill = PatternFill(start_color="FF4444", end_color="FF4444", fill_type="solid")
green_fill = PatternFill(start_color="00E273", end_color="00E273", fill_type="solid")

ws.conditional_formatting.add(
    f"F2:F{ws.max_row}",
    CellIsRule(operator="lessThan", formula=["0"], fill=red_fill)
)
ws.conditional_formatting.add(
    f"F2:F{ws.max_row}",
    CellIsRule(operator="greaterThanOrEqual", formula=["0"], fill=green_fill)
)
```

### Total Row

```python
total_row = ws.max_row + 1
ws.cell(row=total_row, column=1, value="TOTAL")
ws.cell(row=total_row, column=4, value=f"=SUM(D2:D{total_row-1})")
ws.cell(row=total_row, column=5, value=f"=SUM(E2:E{total_row-1})")

# Bold total row
from openpyxl.styles import Font
for cell in ws[total_row]:
    cell.font = Font(bold=True)
```

## File Input: Baca Excel dari User

Kalau user **mengirim/upload file Excel untuk dibaca atau dianalisis** (bukan dibuat), gunakan `markitdown` terlebih dahulu:

```bash
# Convert Excel user ke markdown sebelum analisis
markitdown file_dari_user.xlsx > content.md

# Atau via Python
from markitdown import MarkItDown
md = MarkItDown()
result = md.convert("file_dari_user.xlsx")
print(result.text_content)  # semua sheet → tabel markdown
```

Setelah convert, pass markdown content ke analisis. Lihat: `markitdown` skill untuk detail lengkap.
