# Example: Zuma Performance Analysis v2

**Project:** Zuma YTD 2025 Performance Analysis  
**Date:** 2026-02-16  
**Transformation:** Descriptive deck → Strategic narrative with deep insights

---

## The Challenge

User feedback on v1:
> "ini udah oke, tapi kurang 'joss' aja. kayak oke mostly descriptive, so what? gk ada storytelling (kayak mckinsey gitu) & kurang aja gitu insight analysisnya."

**Problems identified:**
- ✅ Data accurate
- ✅ Visualisasi clean
- ❌ Terlalu descriptive ("Top 20 SKU ini, revenue segini")
- ❌ Kurang storytelling/narrative arc
- ❌ Insights masih shallow ("so what?")

---

## Transformation Strategy

**Applied frameworks:**
1. **SCQA:** Situation → Complication → Question → Answer
2. **Pyramid Principle:** Answer first (Executive Summary)
3. **Insight Hierarchy:** Descriptive → Diagnostic → Predictive → Prescriptive
4. **Narrative Arc:** Challenge → Opportunity → Strategy → Call to Action

---

## Before & After Comparison

### Structure

**❌ V1 (Descriptive):**
```
1. Title
2. Executive Summary (generic highlights)
3. Top 20 SKU (table with data)
4. SKU Insights (generic observations)
5. Branch Performance (revenue by branch)
6. Top 10 Stores (table)

```

**✅ V2 (Narrative Arc):**
```

3. Part 1: The Challenge — WHY declining + Forecast scenarios + Bright spot
4. Part 2: The Opportunity — Pattern discovery (Tourism > Urban, Online dominance)
5. Part 3: The Strategy — 3 plays with SMART recommendations + ROI
6. Implementation Roadmap — Q2-Q4 timeline with milestones
7. Metrics & Risks — KPIs + Risk mitigation
8. Call to Action — Urgency + Deadline
```

## Key Improvements Summary

### 1. Insight Depth

**V1:**  
Descriptive → "YoY decline -83%"

**V2:**  
Descriptive → Diagnostic → Predictive → Prescriptive
- WHAT: -83% decline
- WHY: Product lifecycle + saturation + competition
- FORECAST: Lose Rp 2.3B if unchanged
- ACTION: Launch 10-15 new SKU, invest Rp 150M, ROI 3.3x

### 2. Storytelling

**V1:**  
No narrative arc, feels like disconnected sections

**V2:**  
Clear story:
- **Tension:** Core revenue collapsing
- **Discovery:** Tourism & online are growth engines
- **Resolution:** 3-Play Strategy
- **Urgency:** Deadline Q2 2025, pivot or decline

### 3. Quantification

**V1:**  
Recommendations generic ("launch products", "expand Bali")

**V2:**  
Every recommendation has:
- Specific quantity (10-15 SKU, 5 stores)
- Investment (Rp 150M, Rp 1B, Rp 200M)
- Timeline (Q2, Q3-Q4)
- ROI (3.3x, 14mo payback, 55x)
- Risk mitigation (pilot first, diversify)

### 4. Strategic Coherence

**V1:**  
8 disconnected insights/actions

**V2:**  
3 integrated plays supporting one thesis:
"Pivot from declining core products + urban stores → New products + tourism expansion + digital scale"

## Outcome

**Deployed:** https://zuma-performance-analysis.vercel.app

**User feedback:** TBD (pending review)

**Lesson learned:**  
Data storytelling transforms reports from "interesting" to "actionable". Executives need:
1. Answer upfront (not buried)
2. WHY analysis (not just WHAT)
3. Forecast (what if we don't act?)
4. Specific recommendations with ROI
5. Timeline and accountability
6. Urgency

**Time investment:**  
- Research frameworks: 30 min
- Rebuild deck: 1.5 hours
- Create skill/templates: 1 hour
- Total: 3 hours

**Reusability:**  
This skill + templates now available for all future analyses.
